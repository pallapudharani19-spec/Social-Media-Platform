# 📱 Social Media Platform

A full-stack social media web application that enables users to connect, create posts with images, interact through likes and comments, and manage their profiles. The project is built using modern web technologies and deployed on cloud platforms.

---

## 🚀 Live Demo

*Frontend (Vercel):*  
social-media-platform-eguiysp3b-dharani1905.vercel.app

*Backend API (Render):*  
https://social-media-platform-1-8ssl.onrender.com

---

## ✨ Features

- 🔐 Secure User Authentication using JWT
- 👤 User Sign Up & Login
- ✏️ Edit User Profile
- 📝 Create Text Posts
- 🖼️ Upload Images with Cloudinary
- ❤️ Like & Unlike Posts
- 💬 Add Comments
- 🗑️ Delete Your Own Posts and Comments
- 📱 Responsive User Interface
- ☁️ Cloud-Based Image Storage

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- CSS3
- React Icons

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL (Neon)
- JWT Authentication
- Multer
- Cloudinary

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: Neon PostgreSQL
- Image Storage: Cloudinary

## ⚙️ Installation

### Clone the Repository

bash
git clone https://github.com/pallapudharani19-spec/ColdAlpha-Social-Media-Platform.git


### Backend Setup

bash
cd backend
npm install
npm run dev


### Frontend Setup

bash
cd frontend
npm install
npm run dev


---

## 🔑 Environment Variables

Create a .env file inside the *backend* folder.

env
DATABASE_URL=your_database_url
JWT_SECRET=your_jwt_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret


> *Note:* The values above are placeholders. Never upload your real .env file or API credentials to GitHub.
